О работе

Имею опыт разработки single page application на backbone js, взаимодействия с REST API. Проектирую код, прежде чем писать, не стесняюсь уточнять требования или придумывать новое решение.

Нравится командная работа, для самоорганизации и для взаимодействия с другими участниками работы использую Trello, иногда Redmine. Работаю с git. Экспериментирую в собственных проектах.

Владею Photoshop, Firefox и Sketch в необходимом для программиста уровне.
